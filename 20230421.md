## tuple(튜플)형
- element의 값이 한 번 정해지면 값을 수정 할 수 없다.
- mutable (수정 가능한) list, dict
- immutable (수정 불가한) int(정수), float(소수), str(문자), tuple
- +,* 사용 가능 (연결, 반복 출력)
- ex. a= 10, a+=10 = 20 해당 값 위치로 연결함, (수정이랑 다름)

```python
my_list = [1,2,3]
my_list[0]=5
print(my_list) # [5,2,3]

my_tuple = (1,2,3)
my_tuple[0]= 5 # error
print(my_tuple) # 바뀌면 안 되는 값에 주로 사용
```
