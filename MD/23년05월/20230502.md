## numpy
- numpy란?
- Numerical Python의 약자로 산술계산용 라이브러리
- ndarray(다차원 배열객체) > 빠르고 효율적인 메모리 사용,
- 유연한 브로드스팅

### ndarray란? 
- numpy에서 제공하는 자료구조, N차원의 배열 객체
- 대규모의 데이터 집합을 담을 수 있는 자료구조
- ndarray vs list
- 연속된 메모리 블록에 데이터를 저장
- 같은 종류의 데이터를 담음

### zip( )
- 반복가능한 객체들을(2개 이상) 병렬적으로 묶어주는 함수.
- 각 원소들을 튜플의 형식으로 묶어줌.
