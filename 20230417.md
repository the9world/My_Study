  + Alt+방향키 : 해당 줄을 위 아래로 이동
  + Ctrl+/ : 해당 줄을 주석으로 변경

## Python
- 문자형(String)
- 숫자형(Numeric)
  + 정수형(int) 1, 2, 3 등 소수점이 없는 수
  + 실수형(float) 1.1 1.2 2.1 2.3 등 소수
  + 허수형(complex)
- 연산자(operator)
  - \+ 더하기
  - \- 빼기
  - \* 곱하기
  - / 나누기
  - \*\* 제곱 연산자
  - // 정수 나누기 (몫) 연산자 
  - % 나머지 연산자
## ', "" 사용법

  - \t 탭
    - -print("줄바꿈n\n\t탭t")

  - \' 따옴표
